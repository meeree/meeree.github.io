Methods

SetupWe have inputs \(x_i\) from some space \(D_{in}\), e.g., \(\mathbb{R}^{n_{in} \times n_{timesteps}}\) for recurrent neural networks. We also have parameters \(\theta \in D_{param}\), the parameter space. From this, we define the model state on input \(x_i\) by a mapping $$\phi: D_{in} \times D_{param} \rightarrow D_{state}.$$ This function encapsulates every computation in the model (e.g. in an RNN \(\phi\) is a tuple of the model state at every single time, and the model output at every single time). The full state \(h\) is a tensor superposition of the values of \(\phi\) on all inputs, i.e. a direct sum over the states. Formally, assuming there are B inputs \((x_i)_{i=1}^B\), the state is given by $$h \in (D_{state})^B; \, \, h[i] := \phi(x_i, \theta).$$ Then, define the mean loss $$L := \langle \ell_i(h[i]) \rangle_i.$$ The goal is to find the optimal \(\theta \in D_{param}\) minimizing \(L\) and we do this by iteratively adjusting $$\delta \theta := -\text{d} L / \text{d} \theta.$$ Then, the hidden state evolves according to $$\delta h = -(\frac{\text{d} h}{\text{d} \theta} \frac{\text{d} h}{\text{d} \theta}^T) \cdot \text{Err},$$ where \(\text{Err} := \partial L / \partial h\) is the one-step error signal. The product \(\Phi := \frac{\text{d} h}{\text{d} \theta} \frac{\text{d} h}{\text{d} \theta}^T\) is the extended NTK (eNTK).

Example: Feed-Forward Evolution

Setup Consider a feed-forward network. The function \(\phi\) consists of multiple stacked layers \((\phi[t])_{t=1}^{T-1}\). In particular, $$\phi: \mathbb{R}^{n_{in}} \times D_{param} \rightarrow \mathbb{R}^{n_1} \oplus \mathbb{R}^{n_2} \oplus \cdots \oplus \mathbb{R}^{n_T},$$ here, the state is given by $$h[t+1] := \phi[t](h[t]), \, \, \text{where} \, h[1] = x.$$ So \(n_1 = n_{in}\), the input dimension. Each function \(\phi[t]\) has associated trained weights and biases \(W[t] \in \mathbb{R}^{n_{t+1} \times n_t}\), \(b[t] \in \mathbb{R}^{n_{t+1}}\). Explictly, the vectorized parameter space is a big concatenation over these weights and biases $$D_{param} = \mathbb{R}^{(n_2 \times n_1) + n_2 + (n_3 \times n_2) + n_3 + \cdots + (n_T \times n_{T-1}) + n_T}.$$ Note, assuming \(B\) possible model inputs, here the state \(h\) is a tensor: $$h \in (\mathbb{R}^{n_1} \oplus \mathbb{R}^{n_2} \oplus \cdots \oplus \mathbb{R}^{n_T})^B.$$ We index is like \(h[b, t, i] = \phi(x_b, \theta)[t][i] = (\phi[t](h[b, t-1, :]))[i]\). The final state, \(h[:, T, :]\), is referred to as the output of the network. We define some targets of shape \(y^*[b] \in \mathbb{R}^{n_T}\) and a squared error loss $$\ell_b(h) = \frac{\|h[b, T, :] - y^*[b]\|^2}{B \cdot n_T}.$$ The eNTK \(\Phi\) is a tensor operator, defining how the network activation at every layer and on every single input evolves after one GD step.

Computing the eNTK Firstly, in this case $$\text{Err}[b, T, :] = \partial \ell / \partial h = \frac{2}{B n_T} \cdot (h[b, T, :] - y^*[b]),$$ and for \(t \leq T-1\), $$\text{Err}[b, t, :] = 0, \text{ for } t \leq T-1.$$ i.e., it's a weighted residual on the final layer. Define operators $\mathcal{P}, \mathcal{K}$ defining the multi-step state-state and parameter-state, dependencies, respectively. $$J_\theta[b, t, :] = \frac{\text{d} \phi(x_b, \theta)[t]}{\text{d} \theta}.$$ $$ = \frac{\text{d} (W_{t} h[t-1] + b_[t])}{\text{d} (W_1, b_1, W_2, b_2, \dots, W_{T-1}, b_{T-1}}$$ $$ = vec(0, 0, \dots, h[t-1]^T \otimes Id_{n_t \times n_{t-1}}, 1_{n_t}, 0, \dots, 0).$$ We have $$\delta h[b, t] = J$$
