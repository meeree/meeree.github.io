Introduction

A Neural Implicit Representation (NRep) is a type of model stored by using a neural network for coordinate lookup. Consider a shape \(S \subset \R^d\) (typically \(d = 2\) for two-dimensional shapes or \(3\) for three-dimensional geometric parts). For now, we assume the shape is specified by some occupancy function \(\phi_S : \R^d \rightarrow \{0, 1\}\). Specifically, we have $$\phi_S(x) = \begin{cases} 1 & x \in S \\ 0 & x \not \in S \end{cases}$$ Below, we consider more sophisticated representations (including SDF Signed Distance Fields). Then, an NRep is a parameterized function on some parameter domain \(P\) such that \(\phi : \R^2 \times P \rightarrow \R\) satisfies $$\phi_S(x) \approx \phi(x; \theta),$$ up to some error tolerance (and typially on a disrete set of test points). In recent years, NReps have become a very hot field, with many efficient models proposed by NVidia and others (e.g., NGLoD, Neural Parts). In this blog, I will discuss the unique features that make learning NReps distinct as a task subset.

Model Assumptions

Throughout this work, I will assume the model is given by an autonomous neural ODE with linear input and output embeddings. This assumption just makes the math simpler, but discrete models (e.g., MLPs) are very easy to describe in similar terms. This assumption also matches more recent work, e.g., diffusion generative models. Specifically, assume $$\phi(x; \theta) = w_{out}^T h(x, T; \theta)$$ where for \(t \in (0, T]\), $$\frac{d}{d t} h(x, t; \theta) = f(h(x, t; \theta))$$ with initial condition set by \(x\), $$h(x, 0; \theta) = W_{in} x$$ Here, \(w_{out}\) and \(h(x, T; \theta)\) are vectors in \(\R^n\), \(W_{in}\) is an \(n\) by \(d\) matrix.

Motivating Example

Let's, as usual, start with a very simple motivating example. Let's restrict to \(d = 1\) and assume $$S = (-1, 1) \subset \R$$ is a single interval. Consider the any solution \(\phi(\cdot; \theta)\) of the form above, i.e. a flow with endpoints $$\phi(x; \theta) = w_{out}^T h(x, T; \theta), \quad h(x, 0; \theta) = x w_{in}$$ If it solves the task, \(\phi(x; \theta) = \mathbb{1}_{x \in (-1, 1)}\) is an indicator, so we see the flow \(h\) maps the line defined by \(w_{in}\) in \(\R^n\) to two distinct points: \(0\) for points outside and \(w_{out}\) for points inside \(S\). FTLE Dynamics

Define the readout FLTE mapping, $$\Pi_{x, t} : \delta x \mapsto \delta y(x, t)$$ by $$\Pi_{x,t} = \frac{\d y(x,t)}{\d x} \in \mathbb{R}^{1 \times d}$$ In our case, \(d\) is just \(1\), so we can write lowercase \(\pi_{x,t} \in \R\). Initially, we have \(y(x, 0) = w_{out}^T w_{in} x\), so $$\pi_{x,0} = w_{out}^T w_{in}$$ At the final time \(t = T\), in order to solve the task we have \(y(x, T) = \delta_{x \in S}\), so \(\pi_{x,T}\) is almost zero everywhere, $$|\pi_{x,T}| = \begin{cases} 0 & x \not\in \text{bd} (S) \\ + \infty & x \in \text{bd}(S)\end{cases}$$
